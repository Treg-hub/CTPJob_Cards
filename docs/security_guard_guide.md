# Site Security — Guard Guide (Mobile)

*For Security Guards with the Site Security module enabled*

You use the **Job Cards** mobile app for **gate capture** and **Waste Recovery** field work. You do **not** use CTP Pulse.

---

## Your home screen

When Site Security is enabled for your account, the app opens the **Security** tab by default (or **Waste** if Security is off).

On **Home**, you see a **module hub** — not job-card tiles:

- **Site Security** — gate scans, company cars, visitors
- **Waste Recovery** — when the module is enabled for you

You will **not** see:

- Create Job Card, My Work, View Jobs, or Job History
- A **My Work** tab in the bottom navigation

**Settings** and **Notification Inbox** (bell icon) are still available.

---

## Site Security — daily workflow

1. Open the **Security** tab (or tap **Site Security** on Home).
2. Select the **gate** you are working at.
3. Choose the action:
   - **Vehicle scan in** — disc + driver licence + occupants → **SEC-NNNN**
   - **Vehicle scan out** — scan disc or pick from on-site list
   - **On-foot visitor** — manual entry without a vehicle
   - **Company car** — exit (odometer + purpose) or return (mileage)
4. Use **On site** to see who is currently inside the gate.

> **Company car costs** — only security **managers** and **admins** can add costs on mobile. Guards capture trips only.

---

## Waste Recovery (if enabled)

See the **Waste Recovery Guide** in Documentation for schedule, begin collection, photos, and signature.

Guards:

- Schedule loads and **begin collections**
- Link stock **at collection** via **From stock** (you do not browse the on-site stock inventory list)
- Submit loads — weighbridge and cost review happen on **CTP Pulse**, not on your phone

---

## Permissions

The app still needs **location (Allow all the time)** and **notifications** so off-site alerts can park in your **Notification Inbox** until you are back on site.
