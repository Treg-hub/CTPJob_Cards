# Active Context

This file is an index. The detailed content was split into focused docs to keep things navigable. Existing references to `activeContext.md` remain valid — start here, then drill into the relevant file below.

## Index

- [`currentWork.md`](./currentWork.md) — What's actively in flight right now (Current Work Focus + Next Steps).
- [`recentChanges.md`](./recentChanges.md) — Append-only change log of completed work, newest first.
- [`decisions.md`](./decisions.md) — Active architectural/design decisions and considerations.
- [`learnings.md`](./learnings.md) — Important patterns and preferences, learnings/project insights, current status checklist, and code review summary.

## How to use

- Looking for what we're doing **right now**? → `currentWork.md`
- Need to know what **just shipped** or look up when something changed? → `recentChanges.md`
- Trying to understand **why** a pattern was chosen? → `decisions.md`
- Coding conventions, gotchas, role-based UI rules, status of major efforts? → `learnings.md`

When adding new entries:
- Completed work → prepend to `recentChanges.md` (newest first).
- New in-flight focus item → `currentWork.md`.
- New architectural decision → `decisions.md`.
- New pattern, gotcha, or preference → `learnings.md`.
